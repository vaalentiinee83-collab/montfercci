Connecter Claude Code

Guide pour utiliser Claude Code dans GitHub Codespaces

1

Installe Node.js (si pas déjà fait)

node --version
⚠️ Si la commande échoue, installe Node.js depuis nodejs.org
2

Installe Claude Code via npm

npm install -g @anthropic-ai/claude-code
3

Lance Claude Code dans ton projet

cd ton-projet && claude
✅ Claude va ouvrir un lien de connexion dans ton navigateur pour te connecter avec ton compte Anthropic.
4

Configure ton token GitHub (optionnel)

Pour que Claude puisse interagir avec tes repos GitHub (issues, PRs, etc.) :

export GITHUB_TOKEN=ton_token_ici

Génère un token sur : GitHub → Settings → Developer settings → Personal access tokens

Vérification rapide

claude --version
Si tu vois un numéro de version → Claude Code est bien installé !
